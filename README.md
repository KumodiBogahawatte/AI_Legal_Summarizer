# AI-Generated NLR/SLR Case Summarizer (Prototype - PP1)

This repository contains the 50% prototype for the AI legal summarizer targeting Sri Lankan NLR/SLR judgments.

Sections:
- backend: FastAPI app (OCR, preprocessing, rights detection, summarizer)
- frontend: React demo (upload + view summary)
- data: raw/processed judgment files and Sri Lanka corpus (articles/glossary)
- models: trained artifacts (future)
- docs: API docs & implementation guide

Use SQLite for development (default).
See docs/ for more details.
